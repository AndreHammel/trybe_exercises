# Exercício Bloco 4 - Dia 2 : série 'Vamos Fazer Juntos'

## <font color: red> Descrição: </font>
<br>
Estudo de objetos, array e loops no exercício no qual simula o sorteio da Mega Sena e compara os números de um apostador, dando como resultado o número de acertos. Também o código tem como objetivo analisar várias apostas.

- [x] Uso de Loops como FOR;
- [x] Uso de array e seus métodos como push(), pop() e length;
